# Project Structure

- `pages/` - основные HTML-страницы по разделам проекта
- `styles/` - CSS по тем же разделам
- `scripts/` - JavaScript по разделам и общие скрипты
- `reserve-pages/` - резервные/архивные страницы
- `artifacts/` - вспомогательные файлы, логи и старые артефакты

Основные входные точки:

- `index.html` -> редирект на `pages/home/index.html`
- `pages/home/index.html` -> главная маркетинговая страница
- `pages/home/app-home.html` -> рабочая главная приложения
